# AI Chat Layout Agent

An AI-powered layout editing app.

## Features

- Chat-based layout editing
- Dynamic JSON updates
- Live UI preview
- React frontend
- Node.js backend

## Tech Stack

- React
- Node.js
- Express
- Axios
- CSS

## Installation

### Frontend

```bash
cd client
npm install
npm run dev
```

### Backend

```bash
cd server
npm install
node server.js
```

## Example Commands

- move right
- move left
- increase font
- change color
